# Black Mamba - `1.3.2`

Desktop application for namebase.io. Incomplete but has solid bones for anyone wanting to tinker with it themselves. PRs are welcome :)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

npm run electron:serve
